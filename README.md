# Homework 001 - Object Oriented Programming

## Overview
This project demonstrates the principles of Object Oriented Programming through various examples and exercises. 

## Download Instructions
To download this project, follow these steps:
1. Clone the repository using the command:
   ```
   git clone https://github.com/Yamabiko101/homework_001_OOP.git
   ```
2. Navigate into the project directory:
   ```
   cd homework_001_OOP
   ```

## Testing Guide
To run the tests for this project, ensure you have the necessary dependencies installed. You can run the tests using:
```
# For Python
python -m unittest discover

# For Java
javac *.java
java org.junit.runner.JUnitCore YourTestClass
```

## How It Works
This project consists of several classes and methods that implement key OOP concepts, including inheritance, encapsulation, and polymorphism. The main functionalities are demonstrated through various exercises that show real-world applications of these principles.
